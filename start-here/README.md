# RSTUDIO INSTALLATION

### Software <a href="#software" id="software"></a>

The modules shows how to use RStudio.&#x20;

There are three options to use RStudio:

1. Install R and RStudio to your personal computer
2. Create a free RStudio Cloud account (known as Posit).
3. Using RStudio on a university lab computer
